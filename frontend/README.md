# Aware - PsychoHealth App

Aware — це веб-застосунок для моніторингу психоемоційного стану з елементами гейміфікації. Користувачі можуть вести щоденник (з автоматичним аналізом тональності тексту), проходити щоденні опитування настрою та спостерігати за тенденціями змін свого стану.

---

## Технологічний стек

* **Бекенд:** Python, FastAPI, SQLAlchemy, PostgreSQL
* **Фронтенд:** React, Vite, React Router, Lucide Icons
* **Додатково:** TextBlob, Deep Translator, bcrypt

---

## Основні функції

* **Аутентифікація:** Реєстрація та вхід користувачів (паролі безпечно хешуються).
* **Дашборд:** Відображення рівня здоров'я (HP), досвіду (XP) та днів підряд на платформі.
* **Система квестів:** Щоденні завдання (автоматичні та ручні) для мотивації.
* **Щоденник емоцій:** Збереження текстових записів з AI-аналізом тональності тексту (позитивний, негативний, нейтральний). Якщо рівень стресу критичний, користувач втрачає HP.
* **Опитування:** Швидкі щоденні чекіни для оцінки настрою, сну, енергії та соціальної активності.

---

## Інструкція з локального розгортання

### 1. Налаштування бази даних
1. Відкрийте PostgreSQL.
2. Створіть нову порожню базу даних з назвою `psycho_app`.
3. Змініть логін, пароль та порт у файлі `database.py`, щоб вони збігаються з налаштуваннями вашого локального сервера БД. 


### 2. Запуск Бекенду 
Відкрийте перший термінал у папці з файлами бекенду (там, де лежить `main.py`).

**Крок 2.1: Створення віртуального середовища**
`python -m venv venv`

**Крок 2.2: Активація віртуального середовища**
* На Windows: `venv\Scripts\activate`
* На Mac/Linux: `source venv/bin/activate`

**Крок 2.3: Встановлення залежностей**
Виконайте:
`pip install -r requirements.txt`

**Крок 2.4: Запуск сервера**
`uvicorn main:app --reload`
Сервер запуститься за адресою: `http://127.0.0.1:8000`. При першому запуску SQLAlchemy автоматично створить усі необхідні таблиці в базі даних.

### 3. Запуск Фронтенду 
Відкрийте другий термінал у папці фронтенду (там, де лежить `package.json`).

**Крок 3.1: Встановлення залежностей**
`npm install`

**Крок 3.2: Запуск клієнта**
`npm run dev`
Додаток буде доступний у браузері за адресою: `http://localhost:5173`.
